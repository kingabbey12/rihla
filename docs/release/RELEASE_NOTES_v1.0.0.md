# Release Notes — Rihla v1.0.0

_First public release · build 100 · 27 June 2026_

Welcome to Rihla 1.0 — navigation built for the UAE.

## Highlights
- **Offline-ready navigation** — download emirates and keep navigating without a
  signal (tunnels, underground parking, remote roads).
- **UAE intelligence** — advisory Salik, speed-camera, emirate driving-rule,
  weather, and holiday-traffic awareness.
- **Emergency SOS** — one-tap SOS with an encrypted medical profile and contacts.
- **AI copilot** — natural-language, context-aware journey help.
- **Explore** — fuel, parking, charging, and places nearby.
- **Cloud sync (optional)** — sign in to sync places and preferences across devices.
- **English & Arabic** — full localization and right-to-left layout.
- **Privacy-first** — analytics and crash reporting are off by default; sensitive
  data is encrypted on device.

## Notes
- Advisories (Salik, cameras, driving rules) are informational only and never
  encourage unsafe or illegal driving.
- Emergency tools are a convenience aid, not a replacement for official services
  (UAE: 999 / 998 / 997).
